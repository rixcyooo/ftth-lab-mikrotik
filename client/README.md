# Client Setup

Folder ini berisi konfigurasi dan langkah pengujian client pada lab FTTH.

## File

- `pppoe-client.txt` → konfigurasi dan testing PPPoE client (Ubuntu)
- `hotspot-client.txt` → testing hotspot menggunakan browser berbasis terminal

## Keterangan

Client digunakan sebagai end device untuk melakukan pengujian konektivitas:
- PPPoE client untuk autentikasi melalui PPPoE server
- Hotspot client untuk login melalui captive portal
